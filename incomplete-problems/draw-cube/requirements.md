# draw cube

> cassidoo: May 18th, 2025

Write a function that draws an ASCII art cube of given height `n`.

Example:

```javascript
> drawCube(2)
"
  +----+
 /    /|
+----+ |
|    | +
|    |/
+----+
"

> drawCube(4)
"
   +--------+
  /        /|
 /        / |
+--------+  |
|        |  |
|        |  +
|        | /
|        |/
+--------+
"
```
